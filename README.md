# NAME

App::TimeTracker - time tracking for impatient and lazy command line lovers

# VERSION

version 3.002

# SYNOPSIS

Backend for the `tracker` command. See [tracker](https://metacpan.org/pod/tracker) and/or `perldoc tracker` for details.
